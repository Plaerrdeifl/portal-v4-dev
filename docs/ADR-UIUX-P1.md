# ADR – M4 UI/UX Finish Paket 1

## Status

Umgesetzt als erster visueller Lieferstand auf Basis von `2026.07.15-r7.1.m4-performance-finish-2`.

## Umfang

- Design-Tokens und gemeinsame Oberflächen
- Desktop-Seitenleiste
- mobile Kopfzeile
- mobile Fünf-Punkt-Navigation mit „Mehr“-Dialog
- Vollbild-Startansicht
- Benutzer- und Verbindungsanzeige
- Dashboard als Referenzseite

## Unverändert

Fachlogik, Rollen, Rechte, API-Aufrufe und Apps-Script-Backend bleiben unverändert.

## Logo und Wortmarke

Das vorhandene App-Icon wird in Paket 1 als austauschbarer Platzhalter genutzt. Die Wortmarke verwendet einen Font-Stack mit `Komika Axis` als bevorzugter Schrift, ohne eine Fontdatei auszuliefern. Die endgültige Logo- und Lizenzprüfung erfolgt separat.

## Mobile Navigation

Die vier häufigsten Bereiche werden direkt angezeigt: Dashboard, Fanclub, Aufgaben und Teams. Fanbusse und Administration liegen unter „Mehr“, damit auf iPhone-Breite ausreichend große Touchflächen erhalten bleiben.
