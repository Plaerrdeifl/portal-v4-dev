# Abnahme Phase 2

## 1. Ohne Anmeldung

- PWA öffnet ohne Fehler.
- Status zeigt `Backend bereit`.
- Navigation zeigt Start und Anmeldung.
- Fachbereiche sind nicht sichtbar.

## 2. Google-Login

- `Mit Google anmelden` öffnet die Google-Kontoauswahl.
- Nach erfolgreicher Anmeldung erfolgt die automatische Rückkehr zur GitHub-PWA.
- Der Start zeigt Name, Rolle und Sitzungsende.
- Das Login-Ticket ist danach aus der Adresszeile verschwunden.

## 3. Rechte und Navigation

- Admin sieht Dashboard, Fanclub, Teams, Fanbusse und Admin.
- Reguläre Benutzer sehen nur die vom Backend freigegebenen Bereiche.
- Ein nicht freigegebener Benutzer landet beim Status `Freischaltung beantragt`.

## 4. Echte Daten

- Dashboard öffnen.
- Status wechselt auf `Live verbunden`.
- Kennzahlen entsprechen dem bestehenden Portal.

## 5. Sitzung

- Seite neu laden: Anmeldung bleibt erhalten.
- Abmelden.
- Fachnavigation verschwindet.
- Erneut anmelden funktioniert.

## 6. PWA

- Bestehendes Home-Bildschirm-Symbol einmal schließen/neu öffnen.
- Bei hartnäckigem Cache Safari/Chrome neu laden oder das Symbol neu hinzufügen.
